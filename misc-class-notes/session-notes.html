CSS Display Values

Inline

  • Inline Elements default to the size of their content.
  • It's all in -- line duh



Inline-Block

  • Inline-block elements take the width factor from the inline side, and the height factor from the block side of things.

None

  • Removes all elements from the screen.
  • Helpful to remove items after a user interaction, like crossing off an item from a list.

Block

  • Block elements take up 100% of the viewport.

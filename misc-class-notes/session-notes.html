Flexbox

The container element contains the child elements that are used in the flexbox.
display: flex; designates the container element as a flexbox container.
flex-direction: column or row designates whether the items are aligned along the main axis horizontally, or the cross axis vertically. If you change flex-direction to row, or leave it blank, it will align items horizontally either by row or by default. If you choose column it will switch the main axis to vertical and align everything vertically.
'justify-content: ' aligns things along the main axis. 'align-items' aligns things along the cross axis.

'flex-wrap: wrap' makes it so the items in the container don't crush each other on one line, but instead will fill the next line and the next, etc. If you choose flex-wrap: wrap, you enable 'align-content: ' which allows you to align items along the cross axis. align-content has most of the same values as align-items. You can also use 'gap: ' to add space between all the items in the container. Grid



Example Grid: (Look in actual HTML notes)

Items that fall out of the explicitly defined grid fall into the implicit grid. Implicit grid elements don't inherit any of the values from the explicit grid.

'grid-auto-rows: 100px' defines the size of rows that get added on in the implicit grid.
'grid-auto-flow: column' changes from adding new rows to adding new columns.
grid-auto-columns: 100px' defines the size for each new column that gets added.

By the way, you can mix different types of units together, i.e. 1rem 2em 3fr. REPEAT() and MINMAX() functions (These can be used for any attribute value, and they can be nested too.)
.container {
display: grid;
grid-template-rows: repeat(2, 100px); The repeat function takes two values, how many times you want the value to be repeated, and the second value is the value you want to be repeated.
grid-template-columns: 100px minmax(100px 3fr) 1fr; Minmax function is used to set the minium size and maximum size of an item in a grid cell. }
GRID GAP:
grid-gap: 1rem 2rem, the first value is for rows and the second value is for columns.
